# Discord.py Bot Template

This is a Discord.py bot template, for getting your projects up and running fast.


## Usage
Input your bot token and desired prefix in `settings/settings.json`

Running `main.py` will initialize the bot.
#### Bot initialization
First it will access `settings/settings.json` for the bot token and prefix.

Then it will load every file ending in `.py` in the `/cogs` directory.
